### Ionic Framework
Install Cordova & Ionic globally.
```
$ npm install -g cordova ionic
```

**Create An App**  
---
Blank
```
$ ionic start myApp blank
```
Tabs
```
$ ionic start myApp tabs
```
Sidemenu  
```
$ ionic start myApp sidemenu
```

---

Start the app
```
$ ionic serve
```
